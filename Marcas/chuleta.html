Etiquetas básicas

  • html: Indica el comienzo del documento HTML.
  • head: Contiene información sobre el documento, como el título, los metadatos y los estilos.
  • title: Contiene el título del documento.
  • body: Contiene el contenido principal del documento.

Etiquetas de texto

  • p: Define un párrafo.
  • h1: Define un encabezado de nivel 1.
  • h2: Define un encabezado de nivel 2.
  • h3: Define un encabezado de nivel 3.
  • h4: Define un encabezado de nivel 4.
  • h5: Define un encabezado de nivel 5.
  • h6: Define un encabezado de nivel 6.
  • b: Define el texto en negrita.
  • i: Define el texto en cursiva.
  • u: Define el texto subrayado.
  • strong: Define el texto en negrita, pero con un mayor énfasis que la etiqueta .
  • em: Define el texto en cursiva, pero con un mayor énfasis que la etiqueta .
  • small: Define el texto en un tamaño más pequeño.
  • big: Define el texto en un tamaño más grande.
  • sub: Define el texto como subíndice.
  • sup: Define el texto como superíndice.

Etiquetas de formato

  • br: Inserta un salto de línea.
  • hr: Inserta una línea horizontal.
  • img: Inserta una imagen.
  • a: Define un enlace.

Etiquetas de formularios

  • input: Define un campo de entrada.
  • select: Define un menú desplegable.
  • textarea: Define un área de texto.

Etiquetas de listas

  • ul: Define una lista no numerada.
  • ol: Define una lista numerada.
  • li: Define un elemento de una lista.

Etiquetas de tablas

  • table: Define una tabla.
  • tr: Define una fila de una tabla.
  • th: Define una celda de encabezado de una tabla.
  • td: Define una celda de datos de una tabla.
